# 🐳 Docker Cheatsheet

A quick reference guide covering Docker fundamentals, essential commands, and how Docker fits into a CI/CD pipeline with GitHub Actions and AWS ECR.

## What's Inside

- **Core Concepts** — Docker Engine, Image, Container, Docker Hub, Dockerfile
- **Install Guide** — Windows (WSL required) and Mac
- **Essential Commands** — build, run, push, pull, stop, remove, and more
- **Build & Push Workflow** — step-by-step from local build to Docker Hub
- **CI/CD Flow** — how Docker integrates with GitHub Actions and Amazon ECR
